# Witaj w moim blogu!

To jest mój pierwszy post napisany w **MDX**. Mogę używać wszystkich funkcji Markdown, a także komponenty React!

## Dlaczego MDX?

MDX pozwala na:

- Pisanie w Markdown
- Używanie komponentów React
- Łatwe zarządzanie treścią

```javascript
const hello = () => {
  console.log("Hello World!");
};
```
